# 연산자

## 대입 연산자

```java
int a = 1; // 여기서 '='가 대입 연산자
```

- 변수에 값을 할당할 때 '='를 사용. 이를 대입 연산자라고 함.

## 산술 연산자

- - (더하기)
- - (빼기)
- - (곱하기)
- / (나누기)
- % (나머지)
